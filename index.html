Welcome to PiWatch

PiWatch is a powerful monitoring tool designed to display system information on your Raspberry Pi. It offers a customizable and visually appealing interface similar to Conky, allowing you to keep track of vital system statistics right on your desktop.

Installation Instructions

To get started with PiWatch, follow these steps:

  1. Download the installation script using the link below.
  2. Open a terminal on your Raspberry Pi.
  3. Navigate to the directory where the install.sh script was downloaded.
  4. Make the script executable by running: chmod +x install.sh
  5. Execute the script with: sudo ./install.sh
  6. Follow the on-screen instructions to complete the installation.

Frequently Asked Questions (FAQ)

What is PiWatch?
PiWatch is a system monitoring tool for Raspberry Pi that provides real-time information about your system's performance, similar to Conky.
Do I need to install any dependencies?
PiWatch requires a few standard packages that will be installed automatically by the script. Ensure that your Raspberry Pi is up-to-date before running the installation.
Can I customize PiWatch?
Yes, PiWatch is highly customizable. After installation, you can modify its configuration files to suit your preferences and display the information you care about most.
